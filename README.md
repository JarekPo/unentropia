# Unentropia

Backend for a multi-purpose AI support app for assistance, organising and learning.

## Tech Stack
- FastAPI
- Python
- PostgreSQL

## Features
- AI chat processing
- REST API
- Google authentication

## Roadmap

- Organisation and productivity tools
